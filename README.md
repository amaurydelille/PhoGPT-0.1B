# PhoGPT
English-Vietnamese translator made from a boring afternoon, but since it looked cool I thought I would push it.

Very simple Transformer [(Vaswani and Al, 2017)](https://arxiv.org/abs/1706.03762) of 20M parameters, trained on 255k pairs of sentences (clearly not enough).
